🔧 1. NORMALIZACIÓN DE TEXTO (💥 PRIORIDAD ALTA)

Esto SÍ es un bug real en juegos en español.

Problema:
"Águila" vs "aguila"
"Niño" vs "nino"

👉 rompe validación y fairness

✅ SOLUCIÓN PRO (plug & play)
export function normalizeWord(word: string): string {
  return word
    .trim()
    .toLowerCase()
    .normalize("NFD")
    .replace(/[\u0300-\u036f]/g, "");
}
💡 Mejora extra (muy buena)
.replace(/[^a-zñ]/g, "")

👉 elimina:

espacios raros
símbolos
exploits
⚡ 2. DICCIONARIO → CAMBIO PEQUEÑO, BENEFICIO REAL
