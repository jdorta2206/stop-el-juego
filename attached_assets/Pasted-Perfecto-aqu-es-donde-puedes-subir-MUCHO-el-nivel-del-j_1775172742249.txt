🧠 Problema actual

Tu sistema ahora hace algo tipo:
👉 “preguntar a una IA si una palabra es válida”

❌ Problemas:

IA se equivoca (YouTube ❌)
No es consistente
Puede duplicar categorías
Es lenta
✅ SOLUCIÓN PRO (la que usan juegos buenos)

👉 Usa un sistema híbrido:

1. 📚 Diccionario base (controlado por ti)

Crea listas por categoría:

{
  "fruta": ["manzana", "pera", "tomate"],
  "color": ["rojo", "azul", "verde"],
  "marca": ["youtube", "nike", "apple"]
}

👉 Ventajas:

100% fiable
rápido
sin errores absurdos
2. ⚡ Normalización (CLAVE)

Antes de validar:

minúsculas
quitar acentos
quitar espacios

Ejemplo:

" YouTube " → "youtube"
"Tomaté" → "tomate"

👉 Así evitas errores tontos

3. 🚫 Anti-duplicados

No permitas repetir palabra en otra categoría:

if (respuestasUsadas.includes(palabra)) {
  return "duplicado";
}
4. 🤖 IA solo como respaldo (NO principal)

👉 Usa IA SOLO si:

no está en tu diccionario

Ejemplo:

if (!diccionario[categoria].includes(palabra)) {
   return validarConIA(palabra, categoria);
}
5. 🧠 Reglas especiales (muy importante)

Ejemplo:

“tomate”:
fruta ✅
color ❌ (aunque exista “color tomate”)

👉 Solución:
Define excepciones:

{
  "color": {
    "permitidos": ["rojo", "azul"],
    "bloqueados": ["tomate"]
  }
}
6. ⭐ Sistema de votos (nivel PRO)

Si hay duda:
👉 deja que jugadores voten

👍 válido
👎 inválido

Y guardas resultado:

"youtube": {
  "marca": true,
  "animal": false
}

👉 Esto mejora el sistema con el tiempo

🔥 Resultado final

Con esto consigues:

✅ cero errores absurdos (YouTube)
✅ no duplicados
✅ decisiones consistentes
✅ mejora continua
